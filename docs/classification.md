# Plant Classification Structure

This document outlines a simplified plant classification system used across Florica documentation and internal datasets.

## Taxonomy Levels

- **Kingdom**
- **Division**
- **Class**
- **Order**
- **Family**
- **Genus**
- **Species**

## Functional Categories

These categories are used to group plants in documentation and datasets:

- Trees  
- Shrubs  
- Perennials  
- Annuals  
- Indoor plants  
- Groundcovers  
- Climbers  
- Conifers  

## Notes

This structure is intentionally simplified and used for documentation purposes only.
